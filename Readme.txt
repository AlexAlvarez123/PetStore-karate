* Manual de Ejecución *

1. Ubicar la clase KarateRunner dentro de la carpeta src/test/java
2. En caso se quiera ejecutar todos los casos de prueba asegurarse que el tag que se va ejecutar sea @PetStore
3. En caso solo se quiera ejecutar un caso de prueba en específico verificar los demás tags en el archivo Karate.feature
4. Hacer click derecho en la clase KarateRunner y presionar 'Run KarateRunner'
5. Luego de ejecutar en Runner se podrá observar el reporte en la carpeta target/karate-reports/karate-sumary.html
